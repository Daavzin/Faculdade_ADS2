Comparação de Liguangens

Linguagem Vantagens Desvantagens Opnião
Java portabilidade, orientação a objetos, segurança, grande comunidade e alto desempenho. Complexidade, Memória consumida, Lentidão na inicialização e Sobrecarga de tempo de execução Ainda estou conhecendo melhor java porem tenho um apego muito grande pois o Minecraft foi feito nesta linguagem
C# Integração com Windows, eficiente no gerenciamento do sistema, custo menor e mais seguro em relação a outras linguagens linguagem menos flexível, execução lenta, necessidade de compilar a cada vez que o código for alterado particularmente falando não curti muito a experiência de utilizar esta linguagem no primeiro período
PHP Aprendizado fácil, Código aberto, Suporte de quantidade de dados, Multiplataforma Falta de continuidade, Suporte a Datas, Aplicativos de servidor e Documentação incompleta Não conheço esta linguagem mas me atrai facil aprendizagem dela
Python linguagem interpretada, fácil de Ler, escrever e Aprender, Melhora a Produtividade, Open Source Lento, ruim para Mobile, Erros de Execução, utiliza muita memória Muito conhecida por conta dos vendedores de curso e apesar da facil aprendizagem que foi citada, eu tive dificuldade de instalar as extensões no meu Visual G.
HTML Fácil de usar, Muito útil para iniciantes no campo de web design, Suportado em quase todos os navegadores, estabelecido em quase todos os sites Ele pode criar apenas páginas estáticas e simples, Você deve acompanhar as tags obsoletas e não usá-las, Os recursos de segurança oferecidos pelo HTML são limitados Acredito que por ser a linguagem que eu mais utilizei e tenho conhecimento é hoje a linguagem que mais gosto alem de que é boa para design e seria uma possivel área de atuação para mim
